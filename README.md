## Getting Started

```bash
bun dev
bun run lint
```

# Build app

For web

```bash
bun run build
```

For mobile app Android

```bash
bun run build
bun cap sync
bun cap open android
```

## commit

create branch name:
feature/feature-name
EX: feature/add-nav-bar

bugfix/bug-name
EX: bugfix/navbar-dont-show-up

update/update-name
EX: update/navbar

when finish task, create pull request and select pm(me) as the reviewer
